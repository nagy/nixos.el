#+title: nixos.el - Browse NixOS options and packages
#+author: Daniel Nagy
#+language: en

[[https://github.com/nagy/nixos.el/raw/screenshot/emacs-screenshot.svg]]

nixos.el provides interactive =completing-read= and =tabulated-list-mode=
interfaces for browsing NixOS options and Nix packages from Emacs.

* Features

- Browse NixOS options and packages with =M-x nixos-option= / =nixos-package=
- Tabulated list views (=nixos-browse-options= / =nixos-browse-packages=) with Embark actions
- Local project evaluation (=nixos-package-local=) and tarball URLs (=nixos-package-url=)
- Org link types: =nixos-package:=, =nixos-option:=, =nixos-package-search:=, =nixos-option-search:=
- Marginalia annotations, eldoc, and bookmark support

* Usage

After installing, use =M-x nixos-browse-options= or =M-x nixos-browse-packages=
to browse NixOS options and Nix packages.
Jump directly to a package with =M-x nixos-package=.
